## Project Notes

**macOS Build Fixed - Library Linking Issue Resolved**

The macOS build was failing with dyld errors because media_kit_libs_macos_video has hardcoded Homebrew paths.

**Quick Fix**:
```bash
cd mobile
./fix_macos_libs.sh
open build/macos/Build/Products/Debug/divine.app
```

**For Production Builds**: Need to integrate this into build scripts or Xcode build phases. The script handles:
- Bundling all required dylibs (11 libraries)
- Rewriting library references to @rpath
- Re-signing the app
